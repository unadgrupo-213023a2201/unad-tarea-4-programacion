# Sistema Integral de Gestión de Clientes, Servicios y Reservas

Proyecto realizado en Python utilizando Programación Orientada a Objetos.

## Características

- Abstracción
- Herencia
- Polimorfismo
- Encapsulación
- Manejo avanzado de excepciones
- Registro de logs
- Simulación de operaciones

## Ejecución

Ejecutar:

python main.py
